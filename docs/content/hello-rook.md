This is a placeholder page. It exists to prove the Rook **/learn** content engine
works end to end — a markdown file rendered to server-side HTML that a crawler (or
an AI assistant) can read with **no JavaScript**. Real articles will replace this
content.

## What this is

Rook is an AI fantasy football manager for Yahoo, ESPN, and Sleeper. The articles
published here explain the reasoning behind Rook's valuations and recommendations.

Every figure on a public Rook page comes from real engine records — never invented.
This placeholder deliberately contains no fantasy-football numbers; the real
flagship articles will carry the specifics, with sources.

## Why these pages are server-rendered

The Rook product itself lives behind a login, which AI crawlers cannot see. These
`/learn` pages are public and fully server-rendered, so the reasoning is present in
the raw HTML and can be read and cited without running any JavaScript.
